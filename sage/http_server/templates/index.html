The SaGe SPARQL query server is running!

The SaGe server has a native support for evaluating a subset of the SPARQL syntax. It supports SPARQL queries with the following features:

  • Triple patterns
  • Basic Graph patterns
  • Unions
  • Filters, with simple expressions only (=, !=, <, >=, +, -, etc)
  • GRAPH clauses

To execute complete SPARQL queries using a SaGe server, you need to use a SaGe Smart client. Take a look at existing SaGe Smart Clients to evaluate SPARQL queries on your computer.
